Detailed Description

Receiving information from CMD. Correct extraction and sorting of file data.

Methods: sileStrNumber - number of lines in file, fileStrSize - size of each line in file, fileVectData - values of all lines in file, vectIn - getting CMD options, saveResultNumber - writing the number of results to file, saveResultData - writing results to file.
